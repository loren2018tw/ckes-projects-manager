## Why

部分專案結構相似度高，經常需要以既有專案為模板建立新專案。目前使用者需手動逐一新增任務與設定，耗時且容易遺漏。新增專案複製功能可大幅提升效率，降低人為錯誤。

## What Changes

- 專案列表右側新增「複製」按鈕
- 點擊複製按鈕彈出對話框，輸入新專案名稱並勾選是否複製資源檔
- 複製專案時，完整複製所有任務並保留任務相依關係
- 複製時清空任務的開始日期、截止日期、負責人欄位
- 若勾選複製資源檔，則將所有資源檔案複製到新專案

## Capabilities

### New Capabilities

- `project-clone`: 專案複製功能，包含 UI 操作、任務複製（含相依保留）、資源檔案選擇性複製

### Modified Capabilities

- `project-management`: 新增「複製專案」需求至專案管理規格中

## Impact

- 專案列表頁面新增複製按鈕與對話框元件
- 新增或擴充後端 API 以處理專案複製邏輯
- 任務複製需保留相依結構但清空時間與負責人
- 資源複製需支援 Google Drive 檔案複製
