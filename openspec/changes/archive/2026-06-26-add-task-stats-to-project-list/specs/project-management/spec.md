# project-management Specification (Delta)

## ADDED Requirements

### Requirement: 專案列表顯示各專案任務統計

專案管理頁面在專案清單表格下方，為每個未結案（或依篩選條件）專案獨立顯示任務統計區塊，標題格式為「專案名稱-任務統計」。

#### Scenario: 專案列表顯示任務統計區塊

- **WHEN** 使用者進入專案管理頁面且專案列表載入完成
- **THEN** 頁面在表格下方依序顯示每個專案的任務統計區塊，每個區塊使用 `TaskStatsCard` 元件呈現，標題為「{{專案名稱}}-任務統計」

#### Scenario: 篩選變更時統計同步更新

- **WHEN** 使用者切換篩選條件（進行中/所有專案）
- **THEN** 任務統計區塊同步更新，僅顯示篩選後專案的任務統計

#### Scenario: 載入任務時顯示載入狀態

- **WHEN** 系統正在載入多個專案的任務資料
- **THEN** 每個 TaskStatsCard 顯示「載入中…」狀態

#### Scenario: 各專案任務統計獨立正確

- **WHEN** 所有專案的任務載入完成
- **THEN** 每個專案區塊獨立顯示該專案的任務統計：已完成數、未完成數、已逾期數、即將到期任務列表、進行中任務列表
