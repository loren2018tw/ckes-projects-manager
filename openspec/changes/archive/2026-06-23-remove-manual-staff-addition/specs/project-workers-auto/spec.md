# project-workers-auto Specification

## Purpose

提供專案工作人員自動彙整功能，系統從專案內所有任務的負責人資料中，自動收集、去重後產生工作人員清單，無需管理者手動維護。

## Requirements

### Requirement: 自動彙整專案工作人員

系統每次讀取專案資料時，自動從任務 store 中掃描所有任務的 `assignee` 欄位，收集不重複的 `contact.id`，並從聯絡人 store 中取得對應的姓名與電子郵件。

#### Scenario: 自動收集任務負責人

- **WHEN** 系統需要顯示某專案的工作人員清單
- **THEN** 系統掃描該專案所有任務的 `assignee` 欄位，收集所有不重複的聯絡人 ID，並從聯絡人 store 取得完整資料

#### Scenario: 任務被移除後工作人員自動更新

- **WHEN** 專案內某個任務被刪除或其負責人被移除
- **THEN** 該工作人員仍保留在清單中（若仍有其他任務指派給此人），否則從清單中移除

#### Scenario: 聯絡人資料更新後自動反映

- **WHEN** 某聯絡人在聯絡人管理中更新姓名或電子郵件
- **THEN** 工作人員清單中該聯絡人的資訊同步更新
