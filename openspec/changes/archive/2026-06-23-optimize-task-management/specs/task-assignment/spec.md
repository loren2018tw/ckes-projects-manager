# task-assignment Specification

## Purpose

任務負責人指派功能，讓專案管理者可從專案工作人員清單中選取任務負責人。

## Requirements

### Requirement: 從專案工作人員選取負責人

新增或編輯任務時，負責人欄位以下拉選單顯示該專案的所有工作人員，供使用者選取。

#### Scenario: 顯示專案工作人員清單

- **WHEN** 使用者在新增或編輯任務時點擊「負責人」下拉選單
- **THEN** 選單列出該專案所有工作人員的姓名，並顯示目前已被指派多少任務（以數字標記）

#### Scenario: 選取負責人

- **WHEN** 使用者從下拉選單中選取一位工作人員
- **THEN** 該工作人員的姓名顯示在負責人欄位，任務卡片上顯示該人員的姓名縮寫頭像

#### Scenario: 取消指派負責人

- **WHEN** 使用者點擊負責人欄位的清除按鈕
- **THEN** 負責人欄位清空，任務不指派任何人

### Requirement: 看板卡片顯示負責人

在看板視圖中，每個任務卡片上顯示負責人資訊。

#### Scenario: 有負責人的任務卡片顯示頭像

- **WHEN** 看板中的任務有指派負責人
- **THEN** 任務卡片右上角顯示負責人的姓名縮寫圓形頭像，懸停時顯示完整姓名

#### Scenario: 未指派負責人的任務不顯示頭像區

- **WHEN** 看板中的任務無負責人
- **THEN** 任務卡片右上角不顯示頭像區域，節省版面空間
